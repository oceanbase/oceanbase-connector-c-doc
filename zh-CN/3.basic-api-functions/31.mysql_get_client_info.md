mysql_get_client_info() 
============================================

`mysql_get_client_info()` 返回表示 OceanBase 客户端库版本的字符串。

语法 
-----------------------

```c
const char *
mysql_get_client_info(void)
```



返回值 
------------------------

返回表示 OceanBase 客户端库版本的字符串。

报错信息 
-------------------------

无。
