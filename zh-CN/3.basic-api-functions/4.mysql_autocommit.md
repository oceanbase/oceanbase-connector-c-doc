mysql_autocommit() 
=======================================

`mysql_autocommit()` 用于设置自动提交模式。

语法 
-----------------------

```unknow
my_bool
mysql_autocommit(MYSQL *mysql,
                 my_bool mode)
```



如果 `mode` 为 1，则开启自动提交模式；如果 `mode` 为 0，则关闭自动提交模式。

返回值 
------------------------

如果设置成功，则返回零；如果发生错误，则返回非零值。

报错信息 
-------------------------

无。
