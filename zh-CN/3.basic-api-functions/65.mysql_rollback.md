mysql_rollback() 
=====================================

`mysql_rollback()` 用于回滚当前事务。

语法 
-----------------------

```unknow
my_bool
mysql_rollback(MYSQL *mysql)
```



返回值 
------------------------

执行成功，返回零；如果发生错误，返回非零。

报错信息 
-------------------------

无。

使用说明 
-------------------------

该函数以 `completion_type` 系统变量的值为准。特别是，如果 `completion_type` 的值为 `RELEASE`（或 2），则服务器在终止事务并关闭客户端连接后执行释放。

从客户端程序调用 `mysql_close()` 从客户端侧关闭连接。
