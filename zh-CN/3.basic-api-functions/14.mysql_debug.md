mysql_debug() 
==================================

`mysql_debug()` 使用指定的字符串执行 `DBUG_PUSH`。

语法 
-----------------------

```unknow
void
mysql_debug(const char *debug)
```



返回值 
------------------------

无。

报错信息 
-------------------------

无。

使用说明 
-------------------------

`mysql_debug()` 使用 Fred Fish 调试库。要使用此功能，您必须编译客户端库以支持调试。
