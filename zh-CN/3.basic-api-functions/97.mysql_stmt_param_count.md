mysql_stmt_param_count() 
=============================================

`mysql_stmt_param_count()` 返回 Prepared Statement 中存在的参数数量。

语法 
-----------------------

```unknow
unsigned long
mysql_stmt_param_count(MYSQL_STMT *stmt)
```



返回值 
------------------------

返回一个无符号长整数，表示语句中的参数数量。

报错信息 
-------------------------

无。
