mysql_thread_end() 
=======================================

`mysql_thread_end()` 用于结束使用线程句柄。

语法 
-----------------------

```c
void
mysql_thread_end(void)
```



返回值 
------------------------

无。

使用说明 
-------------------------

在调用 `pthread_exit()` 之前调用此函数以释放由 `mysql_thread_init()` 分配的内存。

因为客户端库不会自动调用 `mysql_thread_end()`，所以必须显式调用它以避免内存泄漏。
