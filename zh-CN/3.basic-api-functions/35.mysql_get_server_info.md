mysql_get_server_info() 
============================================

`mysql_get_server_info()` 返回表示 OceanBase 服务器版本的字符串（例如，"5.6.71"）。

语法 
-----------------------

```c
const char *
mysql_get_server_info(MYSQL *mysql)
```



返回值 
------------------------

返回表示 OceanBase 服务器版本的字符串。

报错信息 
-------------------------

无。
