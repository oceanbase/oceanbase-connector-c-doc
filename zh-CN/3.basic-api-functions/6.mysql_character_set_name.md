mysql_character_set_name() 
===============================================

`mysql_character_set_name()` 返回当前连接的默认字符集名称。

语法 
-----------------------

```unknow
const char *
mysql_character_set_name(MYSQL *mysql)
```



返回值 
------------------------

返回默认字符集名称。

报错信息 
-------------------------

无。
